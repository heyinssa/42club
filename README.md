# 개발 목적

- 모바일 환경에서 spreadsheet의 사용성이 너무 불편

- 많은 사람들이 수정가능하다 보니, 데이터가 잘못 수정 되었을 때, 복구 지점을 잡기 힘듦

<br/>
<br/>

# 핵심기능

- 구글 스프레드 시트 연동

- 동아리 검색

- 동아리 추가

- 기존 동아리 내용 수정
